# xhs_downloads
提供便捷的收集素材的小应用工具

1. 模拟真实浏览器操作，实现登录功能，并且支持自动登录功能。
2. 支持对关键词搜索记录和账号收藏记录自动保存浏览记录功能。
3. 支持通过数据筛选勾选想要的素材，并进行导出选中数据功能，包括文本、图片以及视频等素材。

使用方式（当前仅发布了Win版本）：
1. 下载BrowserTab_V1.01.1121.zip文件
2. 解压BrowserTab_V1.01.1121.zip，打开文件BrowserTab_V1.01.1121，双击BrowserTab.exe启动
3. 登录xhs后，通过关键是搜索或者浏览个人收藏目录
4. 点击数据筛选按钮，通过搜索过滤或者直接勾选数据，选择导出选中即可完成素材下载功能。

声明：本工具无任何商业用途，只是为了提供一个便于大家收集素材的小工具。如果大家还有什么需求，欢迎大家留下问题！
